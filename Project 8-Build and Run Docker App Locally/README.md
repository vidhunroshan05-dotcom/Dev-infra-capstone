

# Project 8 : Build and Run Docker App Locally

## Goal

Practice container deployment by building and running a Dockerized application locally.

---

## Overview

This project demonstrates containerizing a simple Python Flask web application using Docker.

The application was packaged into a Docker image and executed as a container on a local system. Port mapping was used to access the app from the browser.

---

## Technologies Used

* Docker
* Python
* Flask

---

## Application Details

* **Base Image:** python:3.9-slim
* **Application Port:** 5000

---

## Execution Summary

* Created a Dockerfile for the Flask application
* Built the Docker image locally
* Ran the container with port mapping
* Accessed the application via `http://localhost:5000`


---

## Conclusion

This project demonstrates basic Docker concepts such as image creation, container execution, and port mapping. Containerization ensures consistent and portable application deployment.
